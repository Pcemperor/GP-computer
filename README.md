Compute · sharp GPA calculator
A minimalist, privacy-focused GPA calculator designed for Nigerian students. No accounts, no tracking - just fast, clean grade calculation.



 Features
7+ Years Support — Medicine, Engineering, Law — up to 7 years (add more)

Semester-Based — First and second semesters per year

Real-Time Calculation — GPA updates as you type

CGPA Tracking — Automatic overall CGPA across all semesters

What-If Analysis — Simulate future grades and see impact

Performance Charts — Visual GPA trend across semesters

Dark Mode — Easy on the eyes

100% Private — All data stays in your browser (localStorage)

Works Offline — No internet needed after load

Export/Import — Backup your data anytime

Mobile First — Sharp, responsive design

Grading Scale (Nigerian 5-point)
Grade	Point
A	5.0
B	4.0
C	3.0
D	2.0
E	1.0
F	0.0
Quick Start
Clone or download this repository

Open index.html in any modern browser

Start adding your courses

That's it — no setup, no server, no account

bash
git clone https://github.com/Pcemperor/GP-computer.git
cd GP-computer

Built With
HTML5, CSS3, JavaScript

Bootstrap 5 — with sharp corners override

Chart.js — for performance graphs

LocalStorage — for data persistence

 Usage
Click any semester pill (e.g., "Year 1 · 1st sem")

Add your courses, units, and grades

Watch your GPA calculate in real-time

View overall CGPA in the top pill

Check performance trends in the menu

Use "What If?" to plan future semesters

Toggle dark mode from the menu

Privacy
No accounts — Just open and use

No servers — Everything stays in your browser

No tracking — Your data is yours

Export/Import — Full control over your data
